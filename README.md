# Trang Web Giới thiệu Cá nhân

Trang web giới thiệu cá nhân được xây dựng bằng Streamlit Python.

## Cài đặt

1. Cài đặt các thư viện cần thiết:
```bash
pip install -r requirements.txt
```

## Chạy ứng dụng

```bash
streamlit run app.py
```

Ứng dụng sẽ mở tự động trong trình duyệt tại địa chỉ `http://localhost:8501`

## Tùy chỉnh

Bạn có thể chỉnh sửa file `app.py` để:
- Thay đổi thông tin cá nhân
- Cập nhật kinh nghiệm làm việc
- Thêm/bớt kỹ năng
- Thêm dự án mới
- Thay đổi màu sắc và style trong phần CSS

## Cấu trúc

- `app.py` - File chính chứa toàn bộ code của ứng dụng
- `requirements.txt` - Danh sách các thư viện cần thiết
- `README.md` - File hướng dẫn này
